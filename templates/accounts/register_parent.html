    Registration Guide
    Required fields

    First name, last name, gender, phone, relationship to student, and a login password are the minimum required.

    Parent ID

    The system auto-generates a unique ID like PAR20250001. This is the parent's login username. Print it on the admission letter or SMS it to them.

    Linking to students

    After creating this account, link students to this parent from each student's profile page.

    Uganda location

    District, sub-county and village are optional but useful for home-visit records and communications.
